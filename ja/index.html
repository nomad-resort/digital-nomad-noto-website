アクセス

都市部から数時間でアクセスできる、日常から離れた場所

羽田 → 能登里山空港（飛行機）

  1. 羽田 → 能登里山空港：約1時間
  2. 空港 → 七尾市内：車で40〜50分

東京 → 金沢 → 七尾（新幹線＋電車）

  1. 東京 → 金沢：北陸新幹線（約2時間30分）
  2. 金沢 → 七尾：JR七尾線（約60〜80分）

大阪 → 敦賀 → 金沢 → 七尾（特急＋北陸新幹線＋電車）

  1. 大阪 → 敦賀：特急サンダーバード（約1時間20〜30分）
  2. 敦賀 → 金沢：北陸新幹線（約1時間）
  3. 金沢 → 七尾：JR七尾線（約60〜80分）

七尾 → 能登エリア各地（車）

七尾 → 能登島・穴水・珠洲など車で20〜40分。
